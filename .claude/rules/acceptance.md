# Acceptance Criteria Rules

- Before building, map every item in the plan to at least one acceptance criterion
- If a plan item maps to no acceptance criterion, question whether it is in scope
- If an acceptance criterion is not covered by the plan, the plan is incomplete — revise before building
- The PR body must explicitly confirm each acceptance criterion is met, one by one
- Do not mark an AC as met unless it has been verified — not just implemented
